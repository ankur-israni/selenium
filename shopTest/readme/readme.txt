1) Driver Installations
***********************************
Download from: 
Chrome: https://chromedriver.chromium.org/downloads
Firefox: https://github.com/mozilla/geckodriver/releases


Windows
----------
1) chrome: resources/chromedriver.exe
2) firefox: resources/geckodriver_win64.exe
3) IE: resources/IEDriverServer_win64.exe
4) Phantom: resources/phantomjs.exe


Mac
-----
1) chrome: 
brew install chromedriver
Note: The driver will be installed in /usr/local/bin/chromedriver

2) firefox:
brew install geckodriver
Note: The driver will be installed in /usr/local/bin/geckodriver
